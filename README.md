
Auto-BI Modular Studio (Smart Layout 3.0)
----------------------------------------
This modular project contains:
- app.py (entry)
- src/utils.py
- src/llm.py
- src/layout.py (Smart Layout 3.0 + auto-layout)
- src/renderer.py (HTML renderer)
How to run:
- Set OPENAI_API_KEY in Streamlit secrets or env.
- pip install -r requirements.txt
- streamlit run app.py
